# 格式模板

需要写入文件时加载此文件，按需复制对应模板。

---

## 事实卡

写入 `回顾/事实卡/第XXX章.md`。**每条 ≤30 字，全卡 ≤400 字。笔记体，不是句子。**

### 字段模板

```
## 第XXX章 事实卡

### 新出场角色
- {角色名}({主/配/反}, {身份词})   ← 每条 ≤15字
- （本章无新角色则写「无」）

### 角色状态变化
- {角色名}: {A→B}，{原因词}   ← 每条 ≤20字，只记变化
- （无变化写「无」）

### 时间推进
- {起点} → {终点}，{X天/X小时}

### 新增伏笔
- F{章号}-{序号}: {内容}   ← ≤20字
- （无则写「无」）

### 回收伏笔
- F{编号}: {回收方式}   ← ≤15字
- （无则写「无」）

### 关键设定展示/变更
- {设定项}: {一句话}   ← ≤20字
- （无则写「无」）

### 章尾钩子
- {钩子}   ← ≤20字
```

### 写法对照

❌ 句子式（啰嗦）：
```
- 本章新出场了一个叫李明的角色，他是主角的大学同学，属于配角阵营
- 主角从一个没有灵力的普通人状态，因为意外触发祖传玉佩，变成了拥有金手指的修士
```

✅ 笔记式（简洁）：
```
- 李明(配角, 主角大学同学)
- 主角: 无灵力→获得金手指，触发祖传玉佩
```

### 硬约束

- 每条一个 `- `，不写多句
- 字段不需要的内容直接写「无」，不为了好看编
- 角色状态变化只记**本章变化的**，没变的不写
- 伏笔编号全局唯一：F + 章节号 + 序号（F003-1 = 第 3 章第 1 个伏笔）
- 全卡超过 500 字视为不合格，重写

## 章节状态

写入 `回顾/章节状态.md`：

```
## 章节状态追踪

| 章节 | 章名 | 状态 | 字数 | 写完日期 | 发表日期 | 备注 |
|------|------|:----:|------|----------|----------|------|
| 1 | 楔子 | 🔒 | 3200 | 04-20 | 04-21 | |
| 2 | 初入宗门 | ✏️ | 3800 | 04-22 | — | |
```

**原则：** 状态变化立即更新，进入 🔒 填发表日期，备注写需关注的问题。

## 全局状态

写入 `回顾/全局状态.md`：

```
## 全局状态（更新至第XXX章）

### 角色当前状态
| 角色 | 身份 | 当前位置 | 当前动机 | 存活 | 锁定 |
|------|------|----------|----------|------|------|
| {名} | {主/配/反} | {位置} | {动机} | ✅/❓/❌ | 🔒/✏️ |

### 当前时间点
- 故事时间: {年月日/第几天/第几纪元}
- 已推进: {总时间跨度}

### 当前势力/阵营态势
- {势力A}: {状态}

### 活跃主线
- {主线1}: {当前进展}
```

**原则：** 每次归档事实卡后更新，只改变化的部分。

## 伏笔表

写入 `回顾/伏笔表.md`：

```
## 伏笔追踪表

| 编号 | 章节 | 内容 | 状态 | 回收章节 | 回收方式 |
|------|------|------|------|----------|----------|
| F001-1 | 1 | {内容} | ✅已回收 | 5 | {方式} |
| F003-1 | 3 | {内容} | ⏳待回收 | - | - |
| F005-2 | 5 | {内容} | ⚠️疑似遗忘 | - | - |
```

状态标记：⏳待回收 / ✅已回收 / ⚠️疑似遗忘 / ❌已废弃

## 回顾报告

Phase 5 输出用：

```
## 回顾报告（第 X 章 — 第 Y 章）

### 概览
- 审查范围：第 X-Y 章，共 N 章
- 发现矛盾：A 处 / 需修复：B 处
- 伏笔健康度：C/D 正常（E 个待回收，F 个疑似遗忘）

### 矛盾清单

#### 🔴 需立即修复
| # | 章节 | 类型 | 描述 | 修复建议 |
|---|------|------|------|----------|
| 1 | 5, 12 | 人物矛盾 | 李明第5章是同学，第12章变同事 | 统一身份 |

#### 🟡 建议关注
| # | 章节 | 类型 | 描述 | 建议 |
|---|------|------|------|------|

#### 🔒 涉及锁定章节（不可直接修改）
| # | 锁定章节 | 类型 | 描述 | 圆回方案 |
|---|----------|------|------|----------|

### 伏笔状态
- ✅ 正常回收：N 个
- ⏳ 待回收：N 个
- ⚠️ 疑似遗忘：N 个 → 建议在大纲中补充回收计划
```
